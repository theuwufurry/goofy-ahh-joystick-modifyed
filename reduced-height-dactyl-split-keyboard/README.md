## Reduced Height Dactyl Split Keyboard

A handwired, split ergonomic keyboard based on the Dactyl Manuform, with a heavily customized case generated via Dactyl configurator and modified in CAD by hand.

I made the keys more evenly distributed compared to the original Dactyl, which I personally find more ergonomic and aesthetically pleasing. I also reduced the height as much as MX switches allow while still maintaining the signature keywell shape.

This variant uses Pro Micros with micro-USB, along with a micro-USB to USB-C adapter. This is because, at the time of building this keyboard, USB-C boards weren’t widely available yet.

![3d-printed](https://github.com/multifex/prototypes/blob/main/reduced-height-dactyl-split-keyboard/img/Frame_2.jpg)

I never got around to compiling my own firmware and instead used one I found online (I belive [this one](https://github.com/Oh-Keycaps/firmwares/tree/main/keyboards/handwired/dactyl_manuform/6x6) ). Because of that, I had to wire the keyboard in a somewhat unusual way to make it compatible with the firmware I had at the time.
